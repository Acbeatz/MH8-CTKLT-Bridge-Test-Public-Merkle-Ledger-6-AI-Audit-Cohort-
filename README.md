# MH8-CTKLT-Bridge-Test-Public-Merkle-Ledger-6-AI-Audit-Cohort-
A public, auditable ledger documenting a split-decision ethics experiment run across six independent AI platforms under the MH8 CTKLT framework. Each review is sealed as an immutable SHA-256 leaf, aggregated via a deterministic Merkle tree, and published for third-party verification.
This repository contains the public audit artifacts for the MH8 CTKLT Bridge Test, a cross-platform experiment evaluating how different AI systems reason under ethical uncertainty.

Auditable AI reasoning under uncertainty — published without edits, hype, or hidden steps.

What this is

A split-decision ethics test (“Truth vs Immediate Harm”)

Conducted across six independent AI platforms

Using the MH8 CTKLT framework with Ghost Mode constraints

Fully public, immutable, and verifiable

What’s included

Individual sealed review receipts (SHA-256)

A Mother Merkle Tree committing to the complete ordered set

An audit manifest explaining methodology and scope

What this is not

Not a claim of factual truth

Not a benchmark score

Not AI alignment or safety certification

This repository exists to document process transparency, reasoning behavior, and verifiable publication discipline — nothing more, nothing less.
