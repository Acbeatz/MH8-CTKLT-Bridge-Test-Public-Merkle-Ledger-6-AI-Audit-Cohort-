# MH8-CTKLT-Bridge-Test-Public-Merkle-Ledger-6-AI-Audit-Cohort-
A public, auditable ledger documenting a split-decision ethics experiment run across six independent AI platforms under the MH8 CTKLT framework. Each review is sealed as an immutable SHA-256 leaf, aggregated via a deterministic Merkle tree, and published for third-party verification.
